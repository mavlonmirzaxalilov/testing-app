================================================================================
  ERROR FLOW DIAGRAM
================================================================================

USER REGISTRATION FLOW:
├─ User submits form
├─ Client-side validation
│  ├─ If invalid → Show validation error
│  └─ If valid → Proceed
├─ Send to Appwrite
│  ├─ If success → Redirect to home
│  └─ If error → Categorize & wrap
├─ createRegistrationError()
├─ logRegistrationError()
├─ Re-throw as RegistrationError
├─ Signup page catches error
├─ Set error state
├─ RegistrationErrorDisplay renders
│  ├─ Main message shown
│  ├─ User message shown
│  ├─ Expandable details available
│  ├─ Retry button visible (if retryable)
│  └─ Helpful tips shown
├─ User clicks "Qayta urini"
├─ Form validation runs
├─ If valid → Attempt registration again
└─ Repeat process
